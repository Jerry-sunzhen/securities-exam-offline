# 证券从业模拟刷题工具

这是一个依据中国证券业协会公开大纲制作的个人离线学习工具，不是官方题库，也不包含所谓真题或押题。

建议先阅读 [考试说明与资料来源](./exam-guide-and-sources.md) 和 [学习计划](./study-plan.md)。

## 使用方式

1. 使用桌面版 Chrome 打开 `index.html`。
2. 第一次使用时选择“新建学习档案”，创建 `securities-study-profile.sqlite`。
3. Chrome 授权后，答题、收藏、笔记和模考记录会自动写入该文件。
4. 换电脑时，带走这个 SQLite 文件，在另一台电脑中选择“打开学习档案”。

如果浏览器或公司策略不允许网页直接写文件，可使用“导出档案”和“导入档案”。

## 重要提醒

- 换电脑前请确认页面显示“已保存”。
- 不要在两台电脑上同时编辑同一份档案。
- 不要让正在使用的 SQLite 文件由网盘实时同步。
- 本项目只供个人、非商业学习使用。

## 内容基线

- 《证券行业专业人员一般业务水平评价测试大纲（2025）》
- 《证券行业专业人员水平评价测试纪法知识大纲（2026）》
- 题库内容基线和核验截止日期显示在应用侧栏。
- 当前包含 130 个经大纲位置核对的知识单元、390 道题型练习。大纲引用用于定位考试范围；只有标为“答案依据”的引用才表示附有可直接支持答案的原文。

“知识讲义”页面默认按科目和章节展示这 130 个知识单元，包括核心结论、解释、应掌握内容、常见误区和引用来源；“官方大纲原文”作为范围核对入口保留。

## 开发说明

运行工具不需要 Node.js。只有重新生成题库、更新大纲或执行内容校验时才需要开发环境：

```bash
npm install
npm test
```

大纲提取脚本使用 macOS 的 PDFKit/Swift 和 `textutil`；已经生成的发布版不受此限制。
